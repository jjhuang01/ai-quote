# Epic 01 - Evidence-Driven Parity Reconstruction

## Scope

将原插件的可观察能力从“混淆 bundle”转化为“可维护代码 + 可验证测试 + 可追踪文档”。

## Included

- Manifest parity
- Bridge parity（端点级）
- MCP config parity（目标文件级）
- Rules parity（文件级）
- UI parity（入口与状态展示级）

## Excluded for This Iteration

- 未掌握协议细节的远端业务完整语义
- 未证实的隐形账号、积分、License 全量逻辑
